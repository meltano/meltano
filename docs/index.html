Meltano Documentation

Welcome to the Meltano Documentation! Here you'll learn how to use Meltano, how Meltano is built, and where to get started.

To learn how to use Meltano, check out the Getting Started guide and the additional guides and references linked from the Table of Contents in the sidebar.

To learn about contributing to Meltano, refer to the Contributor Guide.

To learn more about the company, you can check our our Company Handbook. This site contains information about our values, mission, vision, and roadmap, all of which guide how we build Meltano out in the open.

Get Help

If you have a question about Meltano, are having trouble getting it to work, or have any kind of feedback, you can:

Check out the Meltano issue tracker to see if someone else has already reported the same issue or made the same request. Feel free to weigh in with extra information, or just to make sure the issue gets the attention it deserves.

Join the Meltano Slack workspace, which is frequented by the core team and thousands of community members and data experts. You can ask any questions you may have in here or just chat with fellow users.
