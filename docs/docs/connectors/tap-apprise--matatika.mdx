# Apprise ERP

## Overview

Apprise is a software solution for businesses. Now acquired by Aptean, Matatika can connect to the Apprise ERP API to produce insights on inventory, sales, purchasing, logistics, and financials, all in one centralized system.

Note: Apprise has been acquired by Aptean. See the [Aptean connector](/connectors/tap-aptean--matatika) for the latest version.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **URIs** | The list of URIs to connect to the Apprise ERP API | String |
