# Trello

## Overview

Trello is a web-based project management tool that allows users to create boards, lists, and cards to organize and prioritize tasks and projects. The platform enables team collaboration through features like member assignment, due dates, attachments, and comments. It integrates with tools such as Google Drive, Slack, and Jira, offering a customizable solution for organizing workflows and improving productivity across teams.

## Prerequisites

**To obtain a Developer API Key and Access Token for Trello:**
1. Go to the [Trello Developer API Key website](https://trello.com/app-key)
2. Log in to your Trello account if you haven't already
3. Click the "Generate API Key" button to generate your Developer API Key
4. Copy the Developer API Key to a safe location
5. Click the "Token" link next to the Developer API Key to generate your Access Token
6. Follow the prompts to authorize the token for your Trello account
7. Copy the Access Token to a safe location

Note: Keep your Developer API Key and Access Token secure, as they provide access to your Trello account.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Developer API Key** | A unique identifier used to authenticate and authorize API requests | Credential |
| **Access Token** | A token that grants access to a user's Trello account and resources | Credential |
| **Start date** | The date from which to start retrieving data from Trello | String |
