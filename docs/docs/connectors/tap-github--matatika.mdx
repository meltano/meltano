# GitHub

## Overview

GitHub is a web-based platform enabling version control and collaboration, providing a centralized location for developers to store and manage their code repositories.

## Prerequisites

**Personal Access Token Setup:**
1. Log in to your GitHub account
2. Navigate to Settings via your profile picture menu
3. Select "Developer settings" from the left sidebar
4. Click "Personal access tokens"
5. Generate a new token with the required scopes
6. Store the token securely — it cannot be retrieved later

**Repository Setup:**
1. Log into GitHub
2. Use the "+" menu to create a "New repository"
3. Configure the repository settings
4. Click "Create repository"

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Personal Access Token** | GitHub personal access token for authentication. | Credential |
| **Repositories** | The repositories to extract data from. | String |
