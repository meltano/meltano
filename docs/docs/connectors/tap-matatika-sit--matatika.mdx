# Matatika SIT

## Overview

Matatika SIT is a test extractor based on tap-spreadsheets-anywhere, used during Matatika SIT (System Integration Test) runs. It is designed for use within the Meltano Cloud platform to support integration testing workflows.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Tables** | Configuration for the tables of data to extract. | Array |
