# Microsoft Access HTTP

## Overview

Microsoft Access is a database management system from Microsoft that combines the relational Access Database Engine with a graphical user interface and software-development tools. This connector enables extraction from Microsoft Access database files (`.mdb` or `.accdb` formats) accessible via HTTP.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Database file** | Local path or URL to a Microsoft Access database `.mdb` or `.accdb` file | String |
| **Connection parameters** | Any parameters for the `fsspec` HTTP storage backend implementation | Object |
