# Aptem

## Overview

Aptem is an end-to-end apprenticeship and vocational training management platform used by training providers and employers to manage learner enrolment, programme delivery, progress tracking, compliance, and end-point assessment.

The connector dynamically discovers streams from the Aptem OData service metadata, exposing 30+ available data streams including learner records, assessments, compliance documents, employment data, and organizational information. Streams with known replication keys sync incrementally; others perform full extractions on each run.

## Prerequisites

**API Access:**
1. Obtain an API token for the Aptem OData API
2. Note your Aptem tenant name, which is used to construct the API base URL
3. Optionally determine a start date for data extraction

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **API Token** | API token for the Aptem OData API | String |
| **Tenant Name** | Aptem tenant name used to build the base URL | String |
| **Start Date** | The earliest record date to sync | Date |
