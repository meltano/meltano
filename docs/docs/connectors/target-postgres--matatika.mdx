# Postgres Warehouse

## Overview

The Postgres connector is a PostgreSQL database loader that functions as a target destination within Meltano's EL (Extract-Load) platform. It enables data loading into PostgreSQL instances with support for multiple load methods, schema flattening, SSH tunnels, and SSL connections.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Host** | Server hostname for the Postgres instance | String |
| **Port** | Connection port number | Integer |
| **User** | Authentication username | String |
| **Password** | Authentication password | Credential |
| **Database** | Target database name | String |
| **Default Target Schema** | Schema designation for incoming data | String |
| **State Schema** | Schema for writing Meltano state | String |
| **Batch Size Rows** | Maximum rows per batch load | Integer |
| **Enable Schema Flattening** | Automatic nested property expansion | Boolean |
| **Max Flattening Depth** | Nesting depth limit for flattening | Integer |
| **Activate Version** | Version message handling toggle | Boolean |
| **Add Record Metadata** | Adds system columns (_sdc_extracted_at, etc.) | Boolean |
| **Hard Delete** | Record deletion vs. soft-delete behavior | Boolean |
| **Interpret Content Encoding** | Schema-based storage optimization | Boolean |
| **Load Method** | append-only, upsert, or overwrite modes | String |
| **Sanitize Null Text Characters** | Chr(0) character handling | Boolean |
| **SSH Tunnel Host** | Address of the bastion host | String |
| **SSH Tunnel Port** | Port to connect to bastion host | Integer |
| **SSH Tunnel Username** | Username to connect to bastion host | String |
| **SSH Tunnel Password** | Password for basic authentication to bastion host | Credential |
| **SSH Tunnel Private Key** | Base64 encoded private key for key pair authentication | Credential |
| **SSH Tunnel Private Key Password** | Private key password (optional) | Credential |
| **Stream Maps** | Custom stream mapping configuration | Object |
| **Use COPY** | Postgres COPY command optimization | Boolean |
| **Validate Records** | Schema validation toggling | Boolean |
| **Faker Locale** | Localization for test data generation | String |
| **Faker Seed** | Deterministic output seeding | Integer |
| **SSL Options** | Certificate and security mode settings | Object |
