# Everflow

## Overview

Everflow is a partner marketing platform that assists businesses in managing and optimizing connections with affiliates, influencers, and other marketing partners.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **API Key** | Everflow network API key. | String |
| **Start Date** | Beginning date for data extraction; defaults to one year prior. | String |
| **User Stream Map Configuration** | Custom configuration values for use within map expressions. | Object |
| **Enable Schema Flattening** | Toggle to activate automatic expansion of nested properties. | Boolean |
| **Max Flattening Depth** | Controls the maximum nesting level for schema expansion. | Integer |
