# Spreadsheets S3

## Overview

Spreadsheets S3 allows users to sync data from spreadsheets stored in Amazon S3 into their chosen targets. The connector supports selecting specific tables and authenticates using AWS credentials.

## Prerequisites

To obtain the Tables required setting for connecting to Spreadsheets Anywhere, you need to have access to the spreadsheet that you want to connect to. Once you have access, you can identify the name of the table or tables that you want to connect to. The table name should be entered in the appropriate field when setting up the connection to Spreadsheets Anywhere.

1. Ensure you have access to the spreadsheet you want to connect to
2. Identify the name(s) of the tables you want to connect to
3. Obtain your AWS Access Key ID and AWS Secret Access Key for authenticating with your S3 bucket

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Tables** | Select which tables to connect to and use in your pipeline | String |
| **AWS Access Key ID** | AWS access key ID for authenticating with your S3 bucket | Credential |
| **AWS Secret Access Key** | AWS secret access key for authenticating with your S3 bucket | Credential |
