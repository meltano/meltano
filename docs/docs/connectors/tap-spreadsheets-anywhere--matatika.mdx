# Spreadsheets Anywhere

## Overview

Spreadsheets Anywhere is a cloud-based software tool that enables users to create and edit spreadsheets on any device with internet access. The connector facilitates real-time collaboration, data import/export, chart creation, and formula customization across desktop, mobile, and tablet devices.

## Prerequisites

**Tables:**
1. Ensure you have access to the spreadsheet you want to connect to
2. Identify the name of the table or tables you want to extract data from

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Tables** | The spreadsheet tables to connect to and use within the application | Array |
