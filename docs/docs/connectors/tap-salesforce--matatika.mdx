# Salesforce

## Overview

Salesforce is a cloud-based customer relationship management (CRM) software that helps businesses manage their sales, marketing, and customer service operations. The platform enables comprehensive customer interaction management across multiple channels including email, social media, phone, and chat. It provides sales teams with lead and opportunity management capabilities, marketing teams with campaign creation tools, and customer service features for issue resolution. The platform includes analytics and reporting functionality to support data-driven business decisions.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Username** | Credentials used to authenticate with the Salesforce API | String |
| **Password** | Authentication credentials for the Salesforce API | Credential |
| **Security Token** | An additional security layer token generated by Salesforce for API authentication | Credential |
| **Start Date** | Specifies the date from which data retrieval from the Salesforce API begins | String |
| **Is Sandbox** | Boolean flag indicating connection to a Salesforce sandbox environment | Boolean |
| **API Type** | Specifies which version of the Salesforce API to use | String |
| **Select Fields By Default** | Boolean setting to auto-select all fields when querying the API | Boolean |
| **State Message Threshold** | Record count threshold for displaying state messages during API queries | Integer |
| **Maximum number of threads to use** | Controls concurrent request capacity for the Salesforce API | Integer |
