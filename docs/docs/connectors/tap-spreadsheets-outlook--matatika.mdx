# Spreadsheets Outlook

## Overview

Sync spreadsheets data from an Outlook mailbox. This connector enables users to extract spreadsheet attachments from Outlook emails using IMAP protocol with glob pattern matching support.

## Prerequisites

**In Outlook:**
1. Identify the file names/formats of email attachments you want to sync

**In Meltano Cloud:**
1. Login and connect with your Microsoft account
2. Add Tables configuration, with respect to the identified files

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **name** | Stream identifier name | String |
| **path** | IMAP folder/email path with glob pattern support (e.g., `imap://imap.outlook.com/INBOX/*/*.csv`) | String |
| **format** | File format: csv, json, jsonl, excel, or detect | String |
| **pattern** | Regex pattern for filename filtering; use `""` if path handles filtering | String |
| **start_date** | ISO-8601 datetime for filtering by last modified timestamp | String |
| **key_properties** | Stream primary keys (e.g., `["_smart_source_bucket", "_smart_source_file", "_smart_source_lineno"]`) | Array |
| **Email** | Mailbox email address | String |
| **encoding** | File encoding (default: utf-8) | String |
| **state_based_discovery** | Enable state-based discovery (default: false) | Boolean |
| **skip_initial** | Lines to skip at start (default: 0) | Integer |
| **max_sampled_files** | Files sampled during discovery (default: 50) | Integer |
| **max_sampling_read** | Lines sampled per file (default: 1000) | Integer |
| **sample_rate** | Sampling frequency (default: 5) | Integer |
| **prefer_schema_as_string** | Skip type inference (default: false) | Boolean |
| **delimiter** | CSV delimiter (default: ,) | String |
| **quotechar** | CSV quote character (default: ") | String |
| **worksheet_name** | Specific Excel worksheet name | String |
