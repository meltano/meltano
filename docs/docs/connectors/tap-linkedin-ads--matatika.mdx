# LinkedIn Ads

## Overview

LinkedIn Ads is a self-service advertising platform that enables businesses to create and display ads to a targeted audience on LinkedIn. It supports multiple ad formats and allows audience targeting based on professional attributes like job title, company size, and industry. Key capabilities include brand awareness building, lead generation, and website traffic driving, along with detailed analytics for campaign measurement and performance optimization.

## Prerequisites

**LinkedIn Ads Account:**
1. Create or log in to a [LinkedIn Ads account](https://www.linkedin.com/campaignmanager/)

**Access Token:**
1. Go to the [LinkedIn Developer portal](https://www.linkedin.com/developers/)
2. Create a new LinkedIn app
3. Authenticate the app with your LinkedIn Ads account
4. Generate an access token

**User Agent:**
1. Prepare a descriptive string identifying the application making API requests (e.g., the name of your application or library)

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Start Date** | The date from which to start retrieving data. | String |
| **LinkedIn API Version** | The LinkedIn API version to use. | String |
