# Microsoft Access Anywhere

## Overview

Microsoft Access Anywhere is an extension of the Microsoft Access connector that enables extraction from Microsoft Access database files (`.mdb` or `.accdb`) hosted anywhere — including local paths, HTTP(S), S3, Azure, and other storage backends supported by `fsspec`. Microsoft Access is a database management system from Microsoft that combines the relational Access Database Engine with a graphical user interface and software-development tools.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Database File** | Local path or URL to a Microsoft Access database `.mdb` or `.accdb` file. | String |
| **Connection Parameters** | Any parameters for the `fsspec` storage backend implementation dictated by the `database_file` URL protocol, such as HTTP(S), S3 or Azure. | Object |
