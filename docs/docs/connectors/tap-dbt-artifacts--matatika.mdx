# dbt Artifacts

## Overview

dbt Artifacts can contain useful metadata. Using this tap you can put that metadata into a target so you can analyze it.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **dbt Target Dir** | The directory where dbt will write compiled SQL files and other artifacts. | String |
