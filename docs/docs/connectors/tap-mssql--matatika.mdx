# MSSQL

## Overview

MSSQL is a relational database management system developed by Microsoft. It enables users to store, manage, and retrieve data efficiently, with support for multiple programming languages, advanced security features, high availability options, and tools for data analysis and reporting.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **User** | The username used to authenticate with the MSSQL API | String |
| **Password** | The password used to authenticate with the MSSQL API | Credential |
| **Host** | The hostname or IP address of the server running the MSSQL API | String |
| **Port** | The port number on which the MSSQL API is listening for incoming connections | Integer |
| **Database** | The name of the database to connect to on the MSSQL server | String |
| **SSH Tunnel Host** | Address of the bastion host, this is the host we'll connect to via SSH | String |
| **SSH Tunnel Port** | Port to connect to bastion host | Integer |
| **SSH Tunnel Private Key** | A base64 encoded private key for authentication to the bastion host w/ key pair auth | Credential |
| **SSH Tunnel Private Key Password** | Private key password, leave None if no password is set | Credential |
| **SSH Tunnel Username** | Username to connect to bastion host | String |
| **SSH Tunnel Password** | Password to connect to bastion host w/ basic auth | Credential |
