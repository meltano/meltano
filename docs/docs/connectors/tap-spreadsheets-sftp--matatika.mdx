# Spreadsheets SFTP

## Overview

Sync spreadsheets data from an SFTP server. This connector enables users to extract spreadsheet data from SFTP server sources and integrate it into data pipelines.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Tables** | Select which tables to connect to and use in your pipeline | String |
