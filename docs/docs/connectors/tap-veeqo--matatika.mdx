# Veeqo

## Overview

Veeqo is a cloud-based inventory and order management software for e-commerce businesses. The connector enables extraction of data from the Veeqo API to support advanced analytics and operational efficiency. Key use cases include automated sales reporting, tax liability reconciliation, and forecasting across multiple data streams including orders, products, inventory, and supplier information.

## Prerequisites

**To obtain an API key for Veeqo:**
1. Log in to your Veeqo account
2. Click on your profile icon in the top right corner of the screen
3. Select "Settings" from the dropdown menu
4. Click on "API" in the left-hand menu
5. Click on the "Generate new API key" button
6. Give your API key a name and select the permissions you want to grant
7. Click on the "Generate API key" button
8. Your new API key will be displayed on the screen

## Configuration

| Setting | Description | Type |
