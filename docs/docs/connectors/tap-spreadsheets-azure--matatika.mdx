# Spreadsheets Azure

## Overview

Spreadsheets Azure is a software tool that allows users to sync data from spreadsheets stored in Azure into their chosen targets.

## Prerequisites

**Tables:**
1. Ensure you have access to the spreadsheet stored in Azure that you want to connect to
2. Identify the name of the table or tables you want to extract data from

**Azure Storage Connection String:**
1. Log in to the [Azure Portal](https://portal.azure.com)
2. Navigate to your Storage Account
3. Go to Security + networking > Access keys
4. Copy the Connection string value to use as your credential

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Tables** | The tables to connect to and use in the application | Array |
| **Azure Storage Connection String** | Connection string providing Azure storage account credentials | Credential |
