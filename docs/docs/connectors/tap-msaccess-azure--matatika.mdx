# Microsoft Access Azure

## Overview

Microsoft Access Azure enables extraction from Microsoft Access database files (`.mdb` or `.accdb`) stored in Azure Blob Storage. Microsoft Access is a database management system from Microsoft that combines the relational Access Database Engine with a graphical user interface and software-development tools. This connector is purpose-built for Azure-hosted Access databases and handles authentication via Azure storage account credentials.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Database File** | Local path or URL to a Microsoft Access database `.mdb` or `.accdb` file. | String |
| **Account Name** | Azure storage account name. | String |
| **Account Key** | Azure storage account access key. | String |
