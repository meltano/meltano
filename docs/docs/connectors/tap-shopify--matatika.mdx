# Shopify

## Overview

Shopify is a cloud-based e-commerce platform that provides businesses with the tools they need to create and manage their online stores. The connector enables data extraction from Shopify to integrate with data pipelines.

## Prerequisites

**Access Token:**
1. Log in to your Shopify account
2. Navigate to the Apps tab in the left sidebar
3. Click "Manage private apps"
4. Create a new private app
5. Enter the app name and developer email
6. Select the Admin API access areas required
7. Save the configuration
8. Retrieve the API key and password to generate your access token

**Store URL:**
1. Log in to your Shopify account
2. Click the Online Store tab
3. Select Domains
4. Your primary domain is listed in the Primary domain section

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Access Token** | A unique code that allows an application to access a Shopify store's data | Credential |
| **Store** | The name or URL of the Shopify store that the application will connect to | String |
| **Start Date** | The date from which the application will start retrieving data from the Shopify store | String |
