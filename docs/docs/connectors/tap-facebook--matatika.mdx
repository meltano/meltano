# Facebook Ads

## Overview

A platform integration enabling businesses to create and manage targeted advertisements across Facebook, Instagram, Messenger, and affiliated platforms.

## Prerequisites

**Account ID:**
1. Log into your Facebook Ads account
2. Click the gear icon in the top right corner and select "Account Settings"
3. Your Account ID will be listed at the top of the page

**Connection:**
1. Click "Connect to Facebook"
2. Log in with your Facebook for Business account
3. Choose the business portfolio and accounts to allow access
4. Save

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Account IDs** | The unique identifiers for the Facebook Ads accounts being connected. | Array |
| **Start Date** | The date from which to start retrieving data. | String |
| **End Date** | The date until which to retrieve data. | String |
| **API Version** | The version of API to use. | String |
| **Custom Insight Reports** | List of custom insight reports. | Array |
| **Limit** | Maximum number of records to retrieve per request. | Integer |
