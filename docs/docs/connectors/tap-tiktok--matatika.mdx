# TikTok Ads

## Overview

TikTok Ads is a social media advertising platform for businesses to promote their products or services on TikTok. The connector enables businesses to create and manage ads with various formats including in-feed ads, brand takeovers, and hashtag challenges. It provides analytics and targeting capabilities to optimize campaigns. The extractor supports standard TikTok Ads streams plus custom basic reports.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Advertiser IDs** | The TikTok advertiser account IDs to sync | Array |
| **Start Date** | The earliest record date to sync | String |
| **Include Deleted** | Include deleted entities in supported streams | Boolean |
| **Lookback** | Number of days prior to today to refetch data | Integer |
| **Custom Basic Report** | List of TikTok basic report definitions to expose as streams | Array |
