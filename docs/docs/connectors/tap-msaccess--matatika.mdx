# Microsoft Access

## Overview

Microsoft Access is a database management system from Microsoft that combines the relational Access Database Engine with a graphical user interface and software-development tools. This connector enables extraction of data from Microsoft Access database files in `.mdb` or `.accdb` formats into your Meltano pipeline.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Database File** | Local path or URL to a Microsoft Access database `.mdb` or `.accdb` file. | String |
