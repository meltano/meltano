# Google Sheets

## Overview

Google Sheets is a cloud-based spreadsheet program that allows users to create and edit spreadsheets online, collaborate with others in real-time, and access their work from anywhere with an internet connection.

## Prerequisites

**OAuth Setup:**
1. Go to the Google Cloud Console and create a new project
2. Enable the Google Sheets API for your project
3. Create OAuth 2.0 credentials, specifying the appropriate scopes for the actions you want to perform with the Sheets API
4. Authenticate the user and obtain consent to access Google Sheets data using the Google Sheets API client libraries or the manual OAuth 2.0 flow
5. Store the resulting Access Token and Refresh Token securely

**Sheet ID:**
1. Open the Google Sheet you want to access
2. Copy the ID from the URL in the address bar — it is the long string of characters between "/d/" and "/edit"

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Sheet ID** | The unique identifier for the Google Sheet to connect to. | String |
| **Stream Name** | An optional designation for the stream being created. | String |
| **Child Sheet Name** | Optional specification of a particular child sheet for loading. | String |
| **Key Properties** | Optional key properties configuration for the child sheet. | Array |
