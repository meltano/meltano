# Spotify

## Overview

Spotify is a digital music service that allows users to access millions of songs, podcasts, and videos from artists all over the world. The connector enables data extraction from Spotify, supporting features like playlists, personalized recommendations, and social sharing capabilities.

Users must create a Spotify Developer account and register an application to obtain the Client ID and Client Secret. The Refresh Token requires obtaining an Access Token first by authorizing the application to access a user's Spotify account through the Spotify Web API.

## Prerequisites

**Client ID and Client Secret:**
1. Create a Spotify Developer account at [developer.spotify.com](https://developer.spotify.com)
2. Log in and navigate to the Dashboard
3. Create a new application
4. Once the application is created, your Client ID and Client Secret will be visible in the application dashboard

**Refresh Token:**
1. Use your Client ID and Client Secret to authorize your application to access a user's Spotify account via the Spotify Web API
2. Complete the OAuth authorization flow to obtain an Access Token
3. Use the Access Token to request a Refresh Token from the Spotify Web API
4. The Refresh Token will be returned in the response

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Client ID** | A unique identifier for your application registered with Spotify | String |
| **Client Secret** | A secret key used to authenticate your application with Spotify | Credential |
| **Refresh Token** | A token used to obtain a new access token when the current one expires | Credential |
