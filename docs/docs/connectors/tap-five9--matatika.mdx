# Five9

## Overview

Five9 is a cloud-based contact center software provider that enables businesses to manage customer interactions across multiple channels including voice, email, chat, and social media. The platform offers automatic call distribution, interactive voice response, predictive dialer, and workforce management tools along with real-time analytics and reporting capabilities.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Username** | Five9 admin username. | String |
| **Password** | Five9 admin password. | Credential |
| **Region** | Five9 region. | String |
| **Start Date** | The date from which to start retrieving data. | String |
| **Custom Reports** | User defined custom reports. | Object |
