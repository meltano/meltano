# Solarvista Live

## Overview

Solarvista Live is a cloud-based software platform designed to help businesses manage their field service operations more efficiently. It offers scheduling, dispatching, work order tracking, and inventory management capabilities with a customizable interface for field service personnel.

## Prerequisites

**Datasources, Account, Client ID, and Code:**
1. Contact your Solarvista Live administrator or IT department to obtain the datasources required to connect
2. Obtain your account credentials from your Solarvista Live administrator
3. Request the client ID specific to your organization from your Solarvista Live administrator
4. Obtain the authentication code specific to your user from your Solarvista Live administrator

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Datasources** | The data sources to connect to in Solarvista Live | Array |
| **Account** | The account name used for authentication | String |
| **Client ID** | The client ID required for authentication | String |
| **Code** | The authentication code | Credential |
| **Start Date** | The date from which data retrieval begins | String |
| **Force Start Date** | Flag to enforce the start date regardless of existing data | Boolean |
