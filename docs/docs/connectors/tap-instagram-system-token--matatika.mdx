# Instagram (System Access Token)

## Overview

Instagram is a social media platform for sharing photos and videos. This connector enables data extraction from Instagram business accounts using system access tokens for authentication, providing access to user profiles, media content, engagement metrics, and insights through the Instagram Graph API. Supported data streams include UsersStream, MediaStream, MediaCommentsStream, MediaInsightsStream, UserInsightsDailyStream, UserInsightsLifetimeStream, and CustomUserInsightReports.

## Prerequisites

**Generating a System Access Token:**
1. Navigate to [Facebook Business](https://business.facebook.com/)
2. Go to **Settings** → **Apps** → **Add** (if the app is not already present). Connect assets if the app lacks account access.
3. Access **Settings** → **System Users** and use the three-dot menu to select **Assign Assets**
4. After assignment, click the **Generate Token** button
5. Select the expiration date and required permissions: `instagram_basic, instagram_manage_insights, pages_show_list, pages_read_engagement`
6. Complete the OTP verification process to receive your system access token

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **IG User IDs** | A unique identifier for an Instagram user account. | String |
| **Access Token** | A token that grants access to the Instagram API. | String |
| **API Version** | The version of the Instagram Graph API to use. | String |
| **Start Date** | The date from which to start retrieving data. | String |
| **Custom User Insight Reports** | List of custom user insight reports. | Array |
