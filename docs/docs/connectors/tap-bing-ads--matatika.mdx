# Bing Ads

## Overview

Bing Ads is a pay-per-click advertising platform that allows businesses to display their ads on Bing and Yahoo search engines. The platform enables organizations to establish and manage ad campaigns with budget controls and performance tracking. It provides targeting capabilities across location, device, and demographic parameters, plus detailed analytics to support optimization of advertising strategies.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **OAuth Client ID** | The OAuth Client ID for client credentials token creation, only used if an oauth2 refresh proxy refresh URL is not supplied. | Credential |
| **OAuth Client Secret** | The OAuth Client Secret for client credentials token creation, only used if an oauth2 refresh proxy refresh URL is not supplied. | Credential |
| **Customer ID** | A unique identifier for the Bing Ads customer account (see cid= in url). | String |
| **Account ID(s)** | A list of unique identifiers for the Bing Ads account(s) associated with the customer account (see aid= in url). | Array |
| **Start Date** | The date from which to retrieve data from the Bing Ads API. | Date |
