# BeautifulSoup

## Overview

BeautifulSoup is a Python library for pulling data out of HTML and XML files. This tap downloads and parses web content, enabling users to scrape websites and convert HTML/XML data into structured formats for data pipelines.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Site URL** | The site you'd like to scrape. The tap will download all pages recursively. | String |
| **Source Name** | The name of the source you're scraping. This will be used as the stream name. | String |
| **Output Folder** | The file path of where to write the intermediate downloaded HTML files to. | String |
| **Parser** | The BeautifulSoup parser engine to use. | String |
| **Download Recursively** | Attempt to download all pages recursively into the output directory prior to parsing files. | Boolean |
| **Exclude Tags** | List of HTML tags to exclude before extracting text content from pages. | Array |
| **Find All Kwargs** | Dictionary containing kwargs for the `find_all` call to extract text from pages. | Object |
| **Flattening Enabled** | Enable schema flattening to automatically expand nested properties. | Boolean |
| **Flattening Max Depth** | Maximum depth for flattening schemas. | Integer |
| **Stream Maps** | Configuration object for stream maps capability. | Object |
| **Stream Map Config** | User-defined configuration values for map expressions. | Object |
