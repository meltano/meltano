# Taboola

## Overview

Taboola Backstage is a content discovery and advertising platform that helps businesses reach their target audience through personalized recommendations and native advertising. The platform functions as a self-service solution enabling businesses to create and manage content discovery and native advertising campaigns. Key capabilities include audience targeting, campaign optimization, real-time analytics, and support for multiple ad formats (video, display, and native ads) across a network of premium publishers.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Client ID** | Taboola Backstage API client ID | String |
| **Client Secret** | Taboola Backstage API client secret | String |
| **Account IDs** | Human-readable identifiers of accounts to extract | String |
| **Start Date** | Initial date to start extracting data from | String |
