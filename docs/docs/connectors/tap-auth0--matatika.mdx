# Auth0

## Overview

Auth0 is a cloud-based platform that provides a comprehensive set of tools and services for managing user authentication and authorization in web and mobile applications. The connector enables developers to add authentication capabilities without building their own identity management system. Key features include social login, multi-factor authentication, passwordless authentication, user management, role-based access control, and third-party identity provider integration.

## Prerequisites

**Auth0 Application Setup:**
1. Access your Auth0 account dashboard
2. Navigate to the Applications section
3. Create a new application, selecting your application type
4. Name your application and complete creation
5. Locate your Client ID and Client Secret in the Settings tab
6. Retrieve your Domain value from your Auth0 account Settings

Note: Implementation steps may vary depending on your Auth0 version.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Client ID** | A unique identifier for the client application that is registered with Auth0. | String |
| **Client Secret** | A secret string that is used to authenticate the client application with Auth0. | Credential |
| **Domain** | The Auth0 domain associated with the tenant. | String |
| **Job Poll Interval Ms** | The interval in milliseconds at which to poll for the status of a long-running job. | Integer |
| **Job Poll Max Count** | The maximum number of times to poll for the status of a long-running job. | Integer |
