# Xero

## Overview

Xero is a cloud-based accounting software for small businesses. The connector enables syncing of financial data including invoicing, bank reconciliation, expense tracking, inventory management, and financial reporting capabilities.

## Prerequisites

To obtain the required settings for connecting to Xero, follow these steps:

1. **Start Date**: The start date is typically the date from which you want to start syncing data from Xero. This can be set based on your specific requirements.
2. **Tenant ID**: The Tenant ID is also known as the Organization ID. You need to have access to at least one Xero organisation.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Tenant ID** | Xero tenant/organisation ID required for authentication | String |
| **Start Date** | Earliest record date to sync in ISO 8601 format | String |
| **Include Archived Contacts** | Optional flag to include archived contacts in the contacts data stream | Boolean |
