# Quickbooks

## Overview

Quickbooks is a cloud-based accounting software that helps small businesses and self-employed individuals manage their finances. The platform enables users to track income and expenses, create invoices, manage bills and payments, and generate financial reports. Additional capabilities include inventory tracking, payroll management, tax preparation, and mobile access.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Realm ID** | The unique identifier for a Quickbooks company | String |
| **Start Date** | The date from which to start retrieving data from Quickbooks in format `%Y-%m-%dT%H:%M:%SZ` | String |
| **Is Sandbox** | A boolean value indicating whether the connection is to a sandbox environment or not | Boolean |
| **Select Fields By Default** | A boolean value indicating whether to select all fields by default when querying Quickbooks data | Boolean |
| **State Message Threshold** | The maximum number of state messages to retrieve from Quickbooks | Integer |
| **Max Workers** | The maximum number of workers to use when processing data from Quickbooks | Integer |
