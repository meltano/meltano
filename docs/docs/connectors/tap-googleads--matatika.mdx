# Google Ads

## Overview

Google Ads is an online advertising platform that allows businesses to create and display ads to potential customers. It functions as a pay-per-click platform enabling businesses to bid on keywords, with features including audience targeting by location, demographics, and interests, plus detailed analytics for campaign optimization.

## Prerequisites

**OAuth Setup:**
1. Obtain the OAuth identity provider authorization endpoint from your identity provider's documentation
2. Determine the required OAuth scopes for the actions you want to perform — refer to the Google Ads API documentation for available scopes
3. Authenticate with Google using OAuth 2.0 to obtain an Access Token
4. Save the OAuth Refresh Token received during initial authentication — it is used to obtain new access tokens when the current one expires

**Developer Token:**
1. Create a Google Ads account if you do not already have one
2. Register for the Google Ads API to obtain a Developer Token, which tracks API usage and ensures policy compliance

**Customer ID:**
1. Log into the Google Ads UI or use the Google Ads API to find the Customer ID, which uniquely identifies each Google Ads account

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Start Date** | ISO start date for all of the streams that use date-based filtering. Defaults to 90 days before the current day. | String |
| **End Date** | ISO end date for all of the streams that use date-based filtering. Defaults to the current day. | String |
| **Login Customer ID** | For manager accounts, set to the manager account's customer ID. | String |
| **Customer IDs** | Sync data for the given customer accounts only. Defaults to all customer accounts discoverable via the authenticated principal. | Array |
| **Enable Click View Report Stream** | Enable the click view report stream (requires certain permissions to be set for your Google Ads account(s)). Not enabled by default. | Boolean |
| **Custom Queries** | List of custom GAQL queries with name, query, date filter option, primary keys, and replication method settings. | Array |
| **API Version** | Specifies which Google Ads API version to use. | String |
