# Baidu

## Overview

Baidu is a leading Chinese technology company providing internet search, cloud computing, AI, and digital advertising services. This connector accesses Baidu's Mediago platform, which manages programmatic advertising campaigns.

The tap provides six data streams:

- **SummaryStream**: High-level campaign summaries
- **CampaignStream**: List of authorized campaigns
- **AccountsStream**: List of authorized accounts
- **CampaignDetails**: In-depth campaign insights
- **ReportInCampaignDimension**: Daily campaign-level reporting
- **ReportInSiteDimension**: Daily site-level reporting for specific campaigns

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **API Token** | Authentication credential for the Mediago API | String |
| **Start Date** | Beginning date for data extraction | Date |
| **End Date** | Ending date for data extraction | Date |
| **Timezone** | Timezone context for the extracted data | String |
