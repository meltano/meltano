# Meltano

## Overview

Meltano is a free and open-source data integration tool that allows users to extract, load, and transform data from various sources into a data warehouse. This connector enables extraction of data from a Meltano instance, providing access to pipeline metadata and project configuration. The platform supports numerous data sources including databases, APIs, and file formats, and features plugins and integrations for popular tools like Google Analytics, Salesforce, and HubSpot.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Meltano Database URI** | The URI for the Meltano database. | String |
