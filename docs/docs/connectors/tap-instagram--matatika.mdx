# Instagram

## Overview

Instagram is a social media platform for sharing photos and videos. This connector enables data extraction from Instagram business accounts, providing access to user profiles, media content, engagement metrics, and insights through the Instagram Graph API. Supported data streams include UsersStream, MediaStream, MediaCommentsStream, MediaInsightsStream, UserInsightsDailyStream, UserInsightsLifetimeStream, and CustomUserInsightReports.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **IG User IDs** | A unique identifier for an Instagram user account. | String |
| **API Version** | The version of the Instagram Graph API to use. | String |
| **Start Date** | The date from which to start retrieving data. | String |
| **Custom User Insight Reports** | List of custom user insight reports. | Array |
