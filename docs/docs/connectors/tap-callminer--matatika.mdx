# CallMiner

## Overview

CallMiner is a speech analytics platform designed to help organizations identify, understand, and optimize customer interactions. The platform leverages artificial intelligence to evaluate customer sentiment and experience metrics.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Client ID** | Credentials for the CallMiner bulk export API client identifier. | Credential |
| **Client Secret** | Authentication credentials for the CallMiner bulk export API secret key. | Credential |
| **Region** | Specifies the CallMiner API geographic location. Options: US, US_FISMA, UK, AU, CA, EU. | String |
| **Start Date** | The initial extraction date for data retrieval, defaulting to the previous year. | Date |
| **Job Poll Max Count** | The maximum number of polling attempts for export job completion (checks every 60 seconds). | Integer |
