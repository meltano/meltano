# Invoca

## Overview

Invoca is an AI-powered revenue execution platform that connects marketing, commerce and contact centre teams. This connector enables data extraction from Invoca into your Meltano pipeline.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Auth Token** | The token to authenticate against the API service. | String |
| **Enable Schema Flattening** | True to enable schema flattening and automatically expand nested properties. | Boolean |
| **Max Flattening Depth** | The max depth to flatten schemas. | Integer |
| **User Stream Map Configuration** | User-defined config values to be used within map expressions. | Object |
