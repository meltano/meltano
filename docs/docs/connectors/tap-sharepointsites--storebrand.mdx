# SharePoint

## Overview

SharePoint is a web-based collaboration and document management platform developed by Microsoft. The connector enables extraction of data from SharePoint for document management, version control, workflow automation, and team collaboration purposes.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Api Url** | The URL of the SharePoint Sites API | String |
| **Lists** | The lists to retrieve data from | Array |
| **Client Id** | The client ID used for authentication | String |
| **Stream Maps** | The mapping of SharePoint fields to output fields | Object |
| **Stream Map Config** | Additional configuration for the stream maps | Object |
| **Flattening Enabled** | Whether or not to flatten nested data structures | Boolean |
| **Flattening Max Depth** | The maximum depth to flatten nested data structures | Integer |
