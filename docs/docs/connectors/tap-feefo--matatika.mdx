# Feefo

## Overview

Feefo is a customer review and feedback management platform that enables businesses to collect, analyze, and display verified customer reviews.

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Merchant ID** | Feefo merchant ID (e.g. `example-retail-merchant`). | String |
| **Client ID** | Feefo app client ID. Used with `client_secret` to access authenticated endpoints, including enriched review data. | String |
| **Client Secret** | Feefo app client secret. Used with `client_id` for authenticated API access with additional data. | Credential |
| **Start Date** | ISO 8601 timestamp specifying the beginning point for data retrieval (inclusive). Defaults to January 1st of the current year. | String |
