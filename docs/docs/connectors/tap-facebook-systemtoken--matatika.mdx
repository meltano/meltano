# Facebook Ads (System Access Token)

## Overview

A platform integration enabling businesses to create and run targeted advertisements on Facebook, Instagram, Messenger, and other affiliated platforms. This variant uses a System Access Token for server-to-server integrations, supporting multiple ad formats and performance tracking capabilities for campaign optimization.

## Prerequisites

**Account ID:**
1. Log into your Facebook Ads account
2. Click the gear icon in the top right corner and select "Account Settings"
3. Your Account ID will be listed at the top of the page

**System Access Token:**
1. Log into your Facebook Developer account
2. Create a new app or select an existing one
3. Log into Facebook for Business
4. Navigate to Settings → Accounts → Apps → Add → Connect an App
5. Go to System Users and add a new system user
6. Click "Generate token" to obtain your System Access Token

## Configuration

| Setting | Description | Type |
|---------|-------------|------|
| **Account IDs** | The unique identifiers for the Facebook Ads accounts being connected. | Array |
| **Access Token** | A token that grants access to the Facebook Ads API. | Credential |
| **Start Date** | The date from which to start retrieving data. | String |
| **End Date** | The date until which to retrieve data. | String |
| **API Version** | The version of API to use. | String |
| **Custom Insight Reports** | List of custom insight reports. | Array |
